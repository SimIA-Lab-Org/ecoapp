# 🩺 EcoApp SimIA: Simulador Ecográfico Remoto

**Laboratorio de Simulación e Innovación de Alicante (SIMIAlab - ISABIAL)**

## 📌 Sobre el Proyecto
EcoApp SimIA es una herramienta de simulación clínica de bajo coste diseñada para democratizar el entrenamiento en ecografía. Permite a los instructores inyectar y controlar imágenes ecográficas (sanas y patológicas) en tiempo real en la pantalla del alumno, utilizando únicamente dispositivos móviles o monitores estándar a través de una red local o conexión web P2P.

## 🚀 Arquitectura Técnica
* **100% Web:** No requiere la instalación de archivos APK ni software de terceros.
* No subir la carpeta node_modules.
* **Comunicación Socket**
* **Sesiones Simultáneas:** Sistema de "salas virtuales" para aislar diferentes escenarios clínicos ocurriendo en tiempo real en el laboratorio.
